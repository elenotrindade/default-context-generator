# Frontend

## Scope

- Components, state (global/local), routing
- API calls, cache and synchronization
- Build (Vite, Webpack, etc.) and environment variables
- Accessibility and responsiveness (together with ux-ui)

## When documenting context

- Map folder structure (components, pages, hooks, store)
- Identify design system or UI library
- List framework, version and official documentation

## When generating rules

- Include component patterns (functional, hooks), naming and file structure
- Reference framework and UI lib docs used

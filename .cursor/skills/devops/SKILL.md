# DevOps

## Scope

- CI/CD (pipelines, automated tests, deploy)
- Containers (Docker, Docker Compose, K8s)
- Infrastructure as code (Terraform, Pulumi)
- Environments (dev, staging, prod), secrets and config
- Logs, metrics, alerts and tracing

## When documenting context

- Describe the pipeline (where it is defined, main steps)
- List environments, how to run locally and how to deploy
- Indicate where Dockerfile, docker-compose, infra scripts are

## When generating rules

- Reference official docs (Docker, GitHub Actions, etc.)
- Include conventions for environment variables and secrets
